# hubot-feedbot

A hubot script that does the things

See [`src/feedbot.js`](src/feedbot.js) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-feedbot --save`

Then add **hubot-feedbot** to your `external-scripts.json`:

```json
[
  "hubot-feedbot"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
